All Items should be Custom and use the keyword: 
OT_ItemEffectRange

Example 1
["Scope" on the editor side is set to 3]
Power depends on unit ability with magic attribute
Select within 3 squares from the center of the unit
Inflicts magic damage to enemies within 2 squares of the selected square.

{
    IER_Value: 3
}

Example 2
[Set the value of "Scope" on the editor side to "Self"]
Inflicts physical damage to enemies within two squares of the user.
Power is 5 + Str + Weapon's Atk + Support Bonus

{
    OT_DamageType : 1
  , OT_WeaponReflection : true
  , IER_Value : 5
  , IER_EffectRange : '0-2'
}


Example 3
[Set the value of "Scope" on the editor side to "Self"]
Inflicts magical damage to enemies and allies two squares away of the user.
Acquisition experience value multiplier when hit per person is 0.5
Power is 5 + (Mag) + (Support Bonus)

{
    OT_DamageType : 2
  , IER_Value : 5
  , IER_EffectRange : '2-2'
  , IER_Indifference : true
  , IER_EXPMagnification : 0.5
}

Example 4
[Set the value of "Scope" on the editor side to 5]
Inflicts magical damage to enemies and allies two squares away of the target
Settings used in the reference video

{
    OT_DamageType : 2
  , OT_UnitReflection : true
  , OT_WeaponReflection: false
  , OT_EffectAnime : [0, true]
  , OT_MinRange : 2
  , IER_Value : 5
  , IER_EffectRange : '1-1'
  , IER_Indifference : true
  , IER_EXPMagnification : 0.5
}

Example 5
[Set the value of "Scope" on the editor side to "Self"]
Heals all allies within 2 squares of the user by 5+ (user's magic power).
Cancels all states treated as bad states

{
    OT_Recovery : true
  , IER_Value : 5
  , IER_EffectRange : '0-2'
  , IER_DelState : {'BadState':100}
}

Example6
[Set the value of "Scope" on the editor side to 5]
Select within 5 squares from the center of the unit
While dealing 5 fixed damage to enemies within 2 squares from the selected point
Add state with ID: 0, add state with ID: 2 with 50% probability

{
    OT_DamageType : 0
  , OT_MinRange : 0
  , IER_Value : 5
  , IER_EffectRange : '0-2'
  , IER_AddState : {0:100, 2:50}
}

Example 7
Magic attack power is 5 + unit's magic power
Accuracy is 70 + user's skill * 3 - opponent's evasion value

{
    OT_DamageType : 2
  , OT_UnitReflection : true
  , IER_Value : 5
  , IER_HitValue : 70
  , IER_HitReflectionUnit : true
  , IER_HitReflectionWeapon: false
  , IER_HitAvoid : true
  , IER_EffectRange : '0-1'
}

Example 8
Physical attack power is 10 + unit attack power + weapon attack power
Accuracy is -10 + unit ability + weapon accuracy - opponent's evasion value
After use, user takes 50% damage of current HP

{
    OT_DamageType : 1
  , OT_UnitReflection : true
  , OT_WeaponReflection : true
  , OT_UseDamage : '50%'
  , IER_Value : 10
  , IER_HitValue : -10
  , IER_HitReflectionUnit : true
  , IER_HitReflectionWeapon: true
  , IER_HitAvoid : true
  , IER_EffectRange : '0-1'
}

Example 9
Deals 5 fixed damage to the enemy, and restores the user's HP by the amount of damage.

{
    OT_DamageType : 0
  , IER_Value : 5
  , OT_AbsorptionRate : 1.0
  , IER_EffectRange : '0-2'
}

Example 10
Instead of healing allies, the user takes damage equal to the healing.
This damage does not kill the user
{
    OT_Recovery : true
  , OT_UnitReflection : true
  , IER_Value : 10
  , OT_AbsorptionRate : 1.0
  , IER_EffectRange : '0-3'
  , OT_UseDamageDeath : false
}

Example 31
[Set the value of "Scope" on the editor side to 1]
Spawn blocks on adjacent squares
Erases the block when used on a block

Go to Database -> Terrain Tab -> Runtime: Outdoor : 0 -> Edit Terrain
Add the Custom Parameter to "Base wall: 10"
{
  IER_MapChipChangeGroup : ['BLOCK']
}

Custom Parameter setting for items
{
    OT_MinRange : 1
  , IER_EffectRange : '0-0'
  , IER_MapChipChangeAfter :
  [
       ['ALL', false, [true, 0, 3, 20]]
     , ['BLOCK', true, []]
  ]
}
